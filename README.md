# auto-scaling-group-project
This project demonstrate how to deploy high available and scaling the web application on using EC2,AUTO SCALING GROUP, classic load balancer or application load balancer, SNS notification and VPC configuration.
# Goal- To automatically scale EC2 instances based on demand and ensure high availability with load balancer
# Key AWS services used: 
# VPC (virtual private cloud)
# EC2 (elastic compute cloud)
# load balancers (classical and application)
# Auto scaling groups
# Simple notification services (SNS)
# Security Groups
